**Multimodal Gemini Agent with Google AI Studio**
An advanced integration engine connecting production codebases to Google AI Studio using the Gemini API. This repository handles multimodal document processing, context management, and integration with live agent architectures.

🚀 Features
Multimodal Ingestion: Pipelines to structure and pass PDFs, images, and audio directly to Gemini models.

Explicit Context Caching: Automated pre-loading of massive reference documents to reduce latency and API overhead cost by up to 80%.

Structured Hybrid Search: Code implementations for combining BM25 exact matching with dense vector retrieval.

Live API Synchronization: Client-side tool calling loops designed for low-latency voice and video streaming agents.
